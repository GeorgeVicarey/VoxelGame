#Voxel Game

The aim of this project is to create a simple Minecraft-Esque game.

###Main goals:

- World made of Cubes
- Camera controlled by 
  * `AWSD` for movement
  * `Mouse` for pitch, yaw and rotation
- Collision detection on cubes


###Pre-Requisites

- SDL2
- GLEW
- SOIL
- OpenGL
